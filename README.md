# Solana Search

## Prerequisite 😈😈

- Node.js (>= 14.x.x)
- npm (>= 6.x.x)
- Rust

## Install

Clone the repository
    git clone https://github.com/voxxx1/sol-search.git

Install the dependencies
    cd sol-search
    npm install

## Run

```
npm start
npm test
docker build -t solana-account-manager .
docker run -it -p 3000:3000 solana-account-manager
```
